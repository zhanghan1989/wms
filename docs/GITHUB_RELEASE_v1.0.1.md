# v1.0.1（2026-02-26）

WMS v1.0.1 是 v1.0.0 后的稳定迭代版本，重点覆盖 FBA 出库链路、检索性能、海外仓编码一致性与运维体验优化。

## 主要更新

- FBA 补货申请一览：
  - 隐藏品牌列；
  - 已出库行展示快递号；
  - 新增“已出库 Excel 下载”（按钮在“出库”右侧）。
- 产品检索性能优化：
  - 精准匹配优先；
  - 关键词 trim；
  - 首页加载路径优化。
- 海外仓管理：
  - 货架号、箱号统一为纯数字规范；
  - 箱号/货架管理弹窗支持固定视窗与滚动增量加载。
- 批量更新库存：
  - 缺失箱号可自动创建并归属默认货架（系统规则）。
- 标签打印：
  - 50x30 标签打印与版式优化；
  - 按实际数量批量打印。
