# WMS 正式版发布说明 v1.0.1

- 发布版本：`v1.0.1`
- 发布日期：`2026-02-26`
- 基线分支：`main`
- 基线标签：`v1.0.1`

## 1. 本次发布内容

- FBA 补货申请一览优化：
  - 隐藏品牌列。
  - 已出库行在操作列显示快递号。
  - 新增已出库数据 Excel 下载按钮（位于“出库”按钮右侧）。
- 海外仓编码规范调整：
  - 货架号、箱号统一为纯数字展示与管理（不再要求 `S-` / `B-` 前缀）。
- 产品检索与首页性能优化：
  - 精准匹配优先、关键词 trim、首页加载路径精简。
- 箱号/货架管理体验优化：
  - 弹窗固定尺寸、默认 10 条、滚动增量加载。
- 批量更新库存增强：
  - 允许自动创建缺失箱号并挂到 `S-00`（按系统当前逻辑）。
- 标签打印能力增强：
  - 支持 50x30 标签打印，条码与文本布局优化，按实际数量打印。

## 2. 兼容性说明

- 无破坏性数据库迁移要求。
- 前端与 API 需同时更新到本版本以使用新增下载能力。

## 3. 验证建议

- `npm run -w api build`
- 核查以下关键流程：
  - FBA 出库、快递号显示、已出库 Excel 下载；
  - 产品检索（精准匹配优先）；
  - 箱号/货架管理弹窗滚动加载；
  - 标签打印与批量更新库存。
