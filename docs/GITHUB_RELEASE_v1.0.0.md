# v1.0.0（2026-02-25）

WMS 第一个正式版发布，覆盖用户权限、海外仓、产品、库存、FBA、批量入库、系统看板、日志与数据备份。

## 主要功能
- 用户与权限管理：登录、用户管理、角色/部门配置。
- 海外仓管理：箱号/货架管理，移动箱子/移动产品。
- 产品管理：单个新增、批量上传、编辑申请、批量确认。
- 库存与FBA：库存调整、FBA补货申请/确认/出库。
- 批量入库：采集箱号、上传校验、模板下载。
- 系统看板：库存健康、需求趋势、备货建议、废弃建议。
- 日志与备份：操作日志、我的日志、数据库备份与ZIP下载。

## 本版关键改进
- erpSKU 变更流程独立规则与权限提示。
- 支持 erpSKU+库存下载、批量更新库存。
- 数据备份支持自动+手动，统一 `Asia/Shanghai` 时区。
- ZIP 下载策略调整为仅保留最近 5 个文件可下载。
