# WMS 正式版发布说明 v1.0.0

- 发布版本：`v1.0.0`
- 发布日期：`2026-02-25`
- 分支：`main`

## 1. 发布范围

本次为第一个正式版，覆盖以下业务域：
- 用户与权限管理
- 海外仓（货架/箱号/移动）
- 产品管理（含编辑申请流程）
- 批量入库与库存管理
- FBA补货流程
- 操作日志/我的日志
- 系统看板
- 数据备份与下载

## 2. 关键功能清单

### 2.1 海外仓管理
- 箱号管理、货架管理（新增/变更/删除）
- 删除前检查与提示（避免直接触发数据库约束报错）
- 移动产品到新箱子支持 SKU 手动输入

### 2.2 产品与库存
- 产品检索结果优化
- rbSKU+库存下载
- 批量更新库存能力
- 批量上传产品能力

### 2.3 批量入库
- 两步式流程：采集箱号 + 上传校验
- 模板下载能力（与 docs 模板文件对齐）

### 2.4 编辑申请
- rbSKU 变更单独规则
- 批量确认
- 详情页确认权限提示

### 2.5 系统看板
- 库存健康
- 出库需求趋势
- 工厂备货建议
- 废弃产品建议
- 废弃箱子建议

### 2.6 备份
- 周日 23:59 自动备份
- 立即备份按钮
- 仅保留最近 5 个 ZIP 文件可下载（历史保留记录）
- 时区统一 `Asia/Shanghai`

## 3. 部署与验证

### 3.1 部署
- 本地/单机：`docker-compose.yml`
- 阿里云部署：`docs/DEPLOY_ALIYUN.md`

### 3.2 验证命令
- `npm run -w api build`
- `npm run -w api lint`
- `npm run -w api test`

## 4. 配套模板文件（docs）

- `批量上传产品.xlsx`
- `批量入库.xlsx`
- `批量更新库存.xlsx`

## 5. 已知事项（本版暂不处理）

- 安全加固项（权限细分、限流、默认密钥治理等）已识别，计划在后续版本处理。
